Project Overview
This project presents an interactive dashboard built using Power BI to analyze a dataset and visualize key insights. The dashboard helps explore patterns, trends, and relationships within the data through clear and interactive visualizations.

Tool Used
Power BI

Dashboard Features
Interactive data visualizations
Trend analysis using charts and graphs
Category-wise comparison
Filters and slicers for dynamic data exploration

Dataset
A structured dataset was used to perform analysis and generate visual insights through Power BI.

Dashboard Preview
Screenshots of the dashboard are available in the images folder.

Objective
The goal of this project is to demonstrate the use of Power BI for transforming raw data into meaningful visual insights through an interactive dashboard.
